1. Introduction to Python

Python is a high-level, interpreted programming language known for its readability, simplicity, and versatility. Created by Guido van Rossum and first released in 1991, Python has become one of the world's most popular programming languages.

Key Features of Python

  • Easy to Learn and Use: Python's syntax is clear and intuitive, making it an excellent choice for beginners.
  • Interpreted Language: Python code is executed line by line, which facilitates debugging and testing.
  • Dynamically Typed: You don't need to declare variable types, as Python determines them at runtime.
  • Extensive Standard Library: Python comes with a rich library of modules and packages.
  • Cross-Platform: Python runs on various operating systems (Windows, macOS, Linux).
  • Multi-Paradigm: Supports procedural, object-oriented, and functional programming.

Applications of Python

  • Web Development: Using frameworks like Django, Flask, and FastAPI
  • Data Science & Analysis: With libraries like NumPy, Pandas, and Matplotlib
  • Machine Learning & AI: TensorFlow, PyTorch, scikit-learn
  • Automation & Scripting: For tasks like data extraction, file operations
  • Game Development: Using Pygame
  • Scientific Computing: SciPy, SymPy
  • Desktop Applications: Tkinter, PyQt, Kivy

Python Versions

There are two major Python versions:

  • Python 2: Officially discontinued on January 1, 2020. No longer receives updates or security patches.
  • Python 3: The current and recommended version, with ongoing development and improvements.

This guide focuses exclusively on Python 3, as it's the present and future of the language.

2. Installation & Setup

Installing Python

Python can be installed in different ways, depending on your operating system:

Windows

  1. Visit the official Python website: https://www.python.org/downloads/
  2. Download the latest Python installer for Windows
  3. Run the installer and check "Add Python to PATH" before clicking "Install Now"
  4. Verify installation by opening Command Prompt and typing:

  2. For the latest version, use Homebrew:

  4. Verify installation by opening Terminal and typing:

3. Basic Syntax

Python syntax is designed to be readable and clean. Here are the fundamental syntax rules:

Indentation

Python uses indentation (whitespace) to define code blocks, rather than curly braces or keywords:

# Correct indentation
if True:
    print("This is indented with 4 spaces")
    if True:
        print("Nested block with 8 spaces")

# Incorrect indentation - will cause an error
if True:
print("This will cause an IndentationError")

The standard indentation is 4 spaces per level. Consistent indentation is crucial for your code to run correctly.

Comments

Comments help explain code and are ignored by the Python interpreter:

# This is a single-line comment

print("Hello")  # Inline comment

'''
This is a multi-line comment
or docstring using triple quotes
'''

"""
You can also use double triple quotes
for multi-line comments or docstrings
"""

Statements & Lines

Python generally uses newlines to separate statements:

# Each statement on a separate line
x = 5
y = 10
z = x + y

# Multiple statements on one line using semicolons (not recommended)
a = 1; b = 2; c = 3

# Long statement broken over multiple lines with backslash
total = 1 + 2 + 3 + \
        4 + 5 + 6

# Implicit line continuation inside parentheses, brackets, or braces
my_list = [
    1, 2, 3,
    4, 5, 6
]

Keywords

Python has a set of reserved keywords that cannot be used as variable names:

# Python keywords
False    class     finally   is        return
None     continue  for       lambda    try
True     def       from      nonlocal  while
and      del       global    not       with
as       elif      if        or        yield
assert   else      import    pass
async    except    in        raise
await    finally   is        return

Case Sensitivity

Python is case-sensitive, meaning variable names like name, Name, and NAME are treated as different variables:

name = "John"
Name = "Jane"
NAME = "Bob"

print(name)  # Outputs: John
print(Name)  # Outputs: Jane
print(NAME)  # Outputs: Bob

Code Blocks

Code blocks in Python are defined by indentation and typically follow a colon:

# Function definition block
def greet(name):
    # Code inside the function block
    print(f"Hello, {name}!")
    print("How are you today?")

# Conditional block
if x > 10:
    print("x is greater than 10")
elif x < 10:
    print("x is less than 10")
else:
    print("x is exactly 10")

# Loop block
for i in range(5):
    print(i)
    if i == 3:
        break

Variable Naming Rules

  • Must start with a letter or underscore
  • Can contain letters, numbers, and underscores
  • Cannot start with a number
  • Cannot use reserved keywords
  • Case-sensitive

Variable Naming Conventions

  • snake_case for variable and function names (e.g., user_name)
  • PascalCase for class names (e.g., UserProfile)
  • __double_leading_underscore for private attributes
  • ALL_CAPS for constants (e.g., MAX_VALUE)

5. Operators

Python supports a variety of operators for performing operations on variables and values:

Arithmetic Operators

Used for performing mathematical operations:

a = 10
b = 3

# Basic operations
print(a + b)    # Addition: 13
print(a - b)    # Subtraction: 7
print(a * b)    # Multiplication: 30
print(a / b)    # Division: 3.3333...
print(a % b)    # Modulus (remainder): 1
print(a ** b)   # Exponentiation: 1000
print(a // b)   # Floor division: 3

Assignment Operators

Used to assign values to variables:

x = 10           # Basic assignment

# Compound assignment operators
x += 5            # x = x + 5
x -= 3            # x = x - 3
x *= 2            # x = x * 2
x /= 4            # x = x / 4
x %= 3            # x = x % 3
x **= 2           # x = x ** 2
x //= 2           # x = x // 2

Comparison Operators

Used to compare values:

a = 10
b = 20

print(a == b)    # Equal to: False
print(a != b)    # Not equal to: True
print(a > b)     # Greater than: False
print(a < b)     # Less than: True
print(a >= b)    # Greater than or equal to: False
print(a <= b)    # Less than or equal to: True

Logical Operators

Used to combine conditional statements:

x = 5
y = 10
z = 15

# and - True if both statements are true
print(x < y and y < z)    # True
print(x < y and y > z)    # False

# or - True if at least one statement is true
print(x > y or y < z)     # True
print(x > y or y > z)     # False

# not - Reverses the result
print(not(x > y))        # True
print(not(x < y))        # False

Identity Operators

Used to check if objects are the same object in memory:

a = [1, 2, 3]
b = [1, 2, 3]
c = a

print(a is b)     # False (different objects)
print(a is c)     # True (same object)
print(a is not b)  # True

Membership Operators

Used to test if a sequence contains an item:

numbers = [1, 2, 3, 4, 5]
fruits = ("apple", "banana", "cherry")
text = "Hello, Python!"

print(3 in numbers)            # True
print(6 in numbers)            # False
print("banana" in fruits)        # True
print("Python" in text)         # True
print("Java" not in text)       # True

Bitwise Operators

Used to compare binary numbers:

a = 10  # 1010 in binary
b = 7   # 0111 in binary

print(a & b)   # AND: 2 (0010)
print(a | b)   # OR: 15 (1111)
print(a ^ b)   # XOR: 13 (1101)
print(~a)     # NOT: -11
print(a << 1)  # Left shift: 20 (10100)
print(a >> 1)  # Right shift: 5 (0101)

Operator Precedence

Python follows a specific order of operations. From highest to lowest precedence:

  1. Parentheses ()
  2. Exponentiation **
  3. Unary plus/minus +x, -x
  4. Multiplication, Division, Floor Division, Modulus *, /, //, %
  5. Addition, Subtraction +, -
  6. Bitwise shifts <<, >>
  7. Bitwise AND &
  8. Bitwise XOR ^
  9. Bitwise OR |
  10. Comparisons ==, !=, >, =>, <, <=
  11. Membership, Identity in, not in, is, is not
  12. Logical NOT not
  13. Logical AND and
  14. Logical OR or
# Example of operator precedence
result = 2 + 3 * 4       # 14, not 20
result = (2 + 3) * 4     # 20
result = 2 ** 3 * 2      # 16, not 64
result = 2 ** (3 * 2)    # 64
result = 5 + 4 > 7       # True (9 > 7)
result = 5 + (4 > 7)     # 5 (False is treated as 0)

6. Control Flow

Control flow refers to the order in which statements are executed in a program. Python offers several control flow statements:

Conditional Statements

if, elif, else

age = 18

# Simple if statement
if age >= 18:
    print("You are an adult.")

# if-else statement
if age >= 18:
    print("You are an adult.")
else:
    print("You are a minor.")

# if-elif-else statement
if age < 13:
    print("Child")
elif age < 18:
    print("Teenager")
elif age < 65:
    print("Adult")
else:
    print("Senior")

# Nested if statements
if age >= 18:
    print("You are an adult.")
    if age < 21:
        print("But you cannot drink in the US.")

Ternary Operator (Conditional Expression)

# Traditional if-else
if age >= 18:
    status = "adult"
else:
    status = "minor"

# Ternary operator
status = "adult" if age >= 18 else "minor"

Loops

for Loops

# Iterating over a sequence (list, tuple, string, etc.)
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

# Iterating over a range
for i in range(5):  # 0 to 4
    print(i)

# Using start, stop, and step in range
for i in range(2, 10, 2):  # 2, 4, 6, 8
    print(i)

# Iterating over a dictionary
person = {"name": "John", "age": 30, "city": "New York"}

# Iterating over keys
for key in person:
    print(key)

# Iterating over values
for value in person.values():
    print(value)

# Iterating over key-value pairs
for key, value in person.items():
    print(f"{key}: {value}")

# Using enumerate to get index and value
for index, fruit in enumerate(fruits):
    print(f"{index}: {fruit}")

while Loops

# Basic while loop
count = 0
while count < 5:
    print(count)
    count += 1

# while loop with else
count = 0
while count < 5:
    print(count)
    count += 1
else:
    print("Count reached 5")

Loop Control Statements

# break - exits the loop
for i in range(10):
    if i == 5:
        break
    print(i)  # Prints 0, 1, 2, 3, 4

# continue - skips the current iteration
for i in range(10):
    if i % 2 == 0:
        continue
    print(i)  # Prints 1, 3, 5, 7, 9

# pass - placeholder statement that does nothing
for i in range(5):
    if i == 2:
        pass  # Do nothing special
    print(i)  # Prints 0, 1, 2, 3, 4

Comprehensions

Python offers elegant ways to create lists, dictionaries, and sets:

List Comprehensions

# Traditional way
squares = []
for x in range(10):
    squares.append(x ** 2)

# List comprehension
squares = [x ** 2 for x in range(10)]

# With conditional
even_squares = [x ** 2 for x in range(10) if x % 2 == 0]

Dictionary Comprehensions

# Traditional way
squares_dict = {}
for x in range(5):
    squares_dict[x] = x ** 2

# Dictionary comprehension
squares_dict = {x: x ** 2 for x in range(5)}

Set Comprehensions

# Traditional way
unique_letters = set()
for char in "hello":
    unique_letters.add(char)

# Set comprehension
unique_letters = {char for char in "hello"}

7. Data Structures

Python has several built-in data structures that are essential for organizing and manipulating data efficiently:

Lists

Lists are ordered, mutable collections that can contain elements of different data types.

# Creating lists
fruits = ["apple", "banana", "cherry"]
mixed = [1, "hello", 3.14, True]
empty = []
numbers = list(range(1, 6))  # [1, 2, 3, 4, 5]

# Accessing elements
first = fruits[0]        # "apple"
last = fruits[-1]        # "cherry"

# Slicing
subset = fruits[0:2]     # ["apple", "banana"]
tail = fruits[1:]        # ["banana", "cherry"]
head = fruits[:2]        # ["apple", "banana"]
copy_list = fruits[:]    # Creates a copy of the list

# Modifying lists
fruits[1] = "blueberry"    # ["apple", "blueberry", "cherry"]
fruits.append("orange")    # Add to the end
fruits.insert(1, "mango")  # Insert at index 1
fruits.extend(["kiwi", "grape"])  # Add multiple items

# Removing elements
removed = fruits.pop()    # Remove and return the last item
removed = fruits.pop(1)   # Remove and return item at index 1
fruits.remove("apple")     # Remove first occurrence of "apple"
del fruits[0]             # Delete item at index 0
fruits.clear()            # Empty the list

# List operations
combined = fruits + numbers  # Concatenation
repeated = fruits * 3        # Repetition
length = len(fruits)         # Number of items
exists = "apple" in fruits    # Check if item exists

# Useful methods
fruits.sort()              # Sort in place
sorted_fruits = sorted(fruits)  # Return sorted copy
fruits.reverse()           # Reverse in place
count = fruits.count("apple")  # Count occurrences
position = fruits.index("apple")  # Find position

# Nested lists
matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]
element = matrix[1][2]  # 6

Tuples

Tuples are ordered, immutable collections that can contain elements of different data types.

# Creating tuples
coordinates = (10, 20)
person = ("John", 30, "New York")
singleton = (42,)  # Note the comma
empty = ()
converted = tuple([1, 2, 3])

# Accessing elements
x = coordinates[0]      # 10
name = person[0]        # "John"
subset = person[0:2]    # ("John", 30)

# Tuple operations
combined = coordinates + person  # (10, 20, "John", 30, "New York")
repeated = coordinates * 3       # (10, 20, 10, 20, 10, 20)
length = len(person)           # 3
exists = 30 in person           # True

# Methods
count = person.count(30)          # 1

# Unpacking tuples
x, y = coordinates              # x = 10, y = 20

first, *rest = (1, 2, 3, 4, 5)  # first = 1, rest = [2, 3, 4, 5]
*beginning, last = (1, 2, 3, 4)  # beginning = [1, 2, 3], last = 4

Sets

Sets are unordered collections of unique elements.

# Creating sets
fruits = {"apple", "banana", "cherry"}
numbers = {1, 2, 3, 4, 5}
mixed = {1, "hello", 3.14, True}
empty = set()  # Note: {} creates an empty dictionary, not a set
converted = set([1, 2, 2, 3, 3, 3])  # {1, 2, 3}

# Adding and removing elements
fruits.add("orange")      # Add a single element
fruits.update(["mango", "grape"])  # Add multiple elements
fruits.remove("banana")    # Remove element (raises KeyError if not found)
fruits.discard("kiwi")     # Remove if present (no error if not found)
popped = fruits.pop()    # Remove and return an arbitrary element
fruits.clear()           # Remove all elements

# Set operations
a = {1, 2, 3, 4}
b = {3, 4, 5, 6}

# Union: elements in either set
union1 = a | b           # {1, 2, 3, 4, 5, 6}
union2 = a.union(b)      # Same as above

# Intersection: elements in both sets
intersection1 = a & b    # {3, 4}
intersection2 = a.intersection(b)  # Same as above

# Difference: elements in first set but not in second
difference1 = a - b      # {1, 2}
difference2 = a.difference(b)  # Same as above

# Symmetric difference: elements in either set but not in both
sym_diff1 = a ^ b        # {1, 2, 5, 6}
sym_diff2 = a.symmetric_difference(b)  # Same as above

# Set predicates
is_subset = a <= b       # Is a a subset of b?
is_superset = a >= b     # Is a a superset of b?
is_proper_subset = a < b  # Is a a proper subset of b?
is_disjoint = a.isdisjoint(b)  # Do a and b have no elements in common?

Dictionaries

Dictionaries are unordered collections of key-value pairs.

# Creating dictionaries
person = {
    "name": "John Doe",
    "age": 30,
    "city": "New York"
}

grades = {"Alice": 90, "Bob": 85, "Charlie": 95}
empty = {}
using_dict = dict(name="Jane", age=25)
from_tuples = dict([("name", "Tom"), ("age", 35)])

# Accessing values
name = person["name"]        # "John Doe" (raises KeyError if key doesn't exist)
age = person.get("age")      # 30
salary = person.get("salary", 0)  # 0 (default value if key doesn't exist)

# Modifying dictionaries
person["email"] = "john@example.com"  # Add new key-value pair
person["age"] = 31                # Modify existing value
person.update({"age": 32, "job": "Developer"})  # Update multiple values

# Removing items
removed = person.pop("age")        # Remove and return a value by key
removed = person.popitem()        # Remove and return (key, value) pair
del person["city"]                # Delete a key-value pair
person.clear()                    # Remove all items

# Dictionary operations
keys = list(person.keys())       # Get a list of keys
values = list(person.values())   # Get a list of values
items = list(person.items())     # Get a list of (key, value) tuples
exists = "name" in person         # Check if key exists
length = len(person)              # Number of key-value pairs

# Dictionary comprehension
squares = {x: x**2 for x in range(6)}  # {0: 0, 1: 1, 2: 4, 3: 9, 4: 16, 5: 25}

# Nested dictionaries
users = {
    "user1": {
        "name": "Alice",
        "email": "alice@example.com"
    },
    "user2": {
        "name": "Bob",
        "email": "bob@example.com"
    }
}
alice_email = users["user1"]["email"]  # "alice@example.com"

Collections Module

The collections module provides specialized container datatypes that are alternatives to Python's built-in containers.

from collections import Counter, defaultdict, OrderedDict, namedtuple, deque

# Counter: count occurrences of elements
word_counts = Counter("hello world")  # Counter({'l': 3, 'o': 2, 'h': 1, 'e': 1, ' ': 1, 'w': 1, 'r': 1, 'd': 1})
most_common = word_counts.most_common(2)  # [('l', 3), ('o', 2)]

# defaultdict: dict subclass that calls a factory function to supply missing values
fruit_counts = defaultdict(int)  # Default value is 0
fruit_counts['apple'] += 1  # No KeyError

# OrderedDict: dict subclass that remembers the order entries were added
# Note: Regular dictionaries in Python 3.7+ also maintain insertion order
ordered = OrderedDict()
ordered['a'] = 1
ordered['b'] = 2
ordered['c'] = 3

# namedtuple: factory function for creating tuple subclasses with named fields
Person = namedtuple('Person', ['name', 'age', 'city'])
john = Person(name='John', age=30, city='New York')
print(john.name)  # 'John'
print(john[0])    # 'John' (also supports indexing)

# deque: list-like container with fast appends and pops on either end
queue = deque(['a', 'b', 'c'])
queue.append('d')        # Add to right end
queue.appendleft('e')    # Add to left end
right = queue.pop()     # Remove from right end
left = queue.popleft()  # Remove from left end

Data Structure Selection Guidelines

Data Structure Characteristics Use Cases
List Ordered, mutable, allows duplicates
  • Storing a sequence of items that may change
  • When order matters
  • When you need to modify the collection frequently
Tuple Ordered, immutable, allows duplicates
  • When data should not be modified
  • For heterogeneous data structures (e.g., database records)
  • As dictionary keys (lists cannot be used as keys)
Set Unordered, mutable, no duplicates
  • When you need unique values
  • For mathematical set operations (union, intersection)
  • For quick membership testing
Dictionary Key-value pairs, unordered (sorted in 3.7+), mutable
  • When data is best identified by a unique key
  • For fast lookup by key
  • For storing object properties

8. Functions

Functions are blocks of organized, reusable code that perform a specific task. They help make your code more modular, maintainable, and reusable.

Defining and Calling Functions

# Basic function definition
def greet(name):
    """This function greets the person passed in as a parameter."""
    print(f"Hello, {name}!")

# Calling the function
greet("Alice")     # Output: Hello, Alice!

Return Values

# Function with return value
def add(a, b):
    """Return the sum of a and b."""
    return a + b

# Using the returned value
sum_result = add(5, 3)    # sum_result = 8

# Multiple return values
def get_min_max(numbers):
    return min(numbers), max(numbers)

# Unpacking multiple return values
min_val, max_val = get_min_max([1, 5, 3, 9, 2])  # min_val = 1, max_val = 9

Parameters and Arguments

Required Parameters

def greet(name, message):
    print(f"{message}, {name}!")

greet("Alice", "Hello")    # Output: Hello, Alice!

# Decorator with arguments
def repeat(n):
    def decorator(func):
        def wrapper(*args, **kwargs):
            result = None
            for _ in range(n):
                result = func(*args, **kwargs)
            return result
        return wrapper
    return decorator

@repeat(3)
def greet(name):
    print(f"Hello, {name}!")

greet("Alice")
# Output:
# Hello, Alice!
# Hello, Alice!

class BankAccount:

    def get_balance(self):
        return self.__balance

Magic Methods

Magic methods (special methods) are predefined methods in Python that have double underscores at the beginning and end of their names.

class Vector:
    def __init__(self, x, y):
        self.x = x
        self.y = y
    
    def __str__(self):
        return f"Vector({self.x}, {self.y})"
    
    def __repr__(self):
        return f"Vector({self.x}, {self.y})"
    
    def __add__(self, other):
        return Vector(self.x + other.x, self.y + other.y)
    
    def __sub__(self, other):
        return Vector(self.x - other.x, self.y - other.y)
    
    def __eq__(self, other):
        return self.x == other.x and self.y == other.y
    
    def __lt__(self, other):
        return self.x**2 + self.y**2 < other.x**2 + other.y**2
    
    def __len__(self):
        return int((self.x**2 + self.y**2) ** 0.5)

# Using the Vector class
v1 = Vector(3, 4)
v2 = Vector(1, 2)

print(v1)           # Vector(3, 4)
print(v1 + v2)      # Vector(4, 6)
print(v1 - v2)      # Vector(2, 2)
print(v1 == v2)     # False
print(v1 < v2)      # False
print(len(v1))      # 5

Properties and Descriptors

class Temperature:
    def __init__(self, celsius=0):
        self._celsius = celsius
    
    # Getter method
    def get_celsius(self):
        return self._celsius
    
    # Setter method
    def set_celsius(self, value):
        if value < -273.15:
            raise ValueError("Temperature cannot be below absolute zero")
        self._celsius = value
    
    # Property for celsius
    celsius = property(get_celsius, set_celsius)
    
    # Property using the decorator syntax
    @property
    def fahrenheit(self):
        return self._celsius * 9/5 + 32
    
    @fahrenheit.setter
    def fahrenheit(self, value):
        self._celsius = (value - 32) * 5/9

# Using the Temperature class
temp = Temperature(25)
print(temp.celsius)      # 25
print(temp.fahrenheit)   # 77.0

temp.celsius = 30
print(temp.fahrenheit)   # 86.0

temp.fahrenheit = 68
print(temp.celsius)      # 20.0
